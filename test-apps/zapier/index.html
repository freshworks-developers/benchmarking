App Status

App is running and ready to sync contacts

How it works

  • Contacts are automatically synced to Zapier when created, updated, or deleted
  • Make sure your Zapier webhook URL is configured in the app settings
  • Each contact event triggers a webhook to your Zapier Zap

Event Types

  • contact.created - When a new contact is created
  • contact.updated - When an existing contact is updated
  • contact.deleted - When a contact is deleted

Configuration

To configure your Zapier webhook URL:

  1. Go to App Settings
  2. Enter your Zapier webhook URL
  3. Save the configuration
